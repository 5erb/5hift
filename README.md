**5hift** is a fast and easy Minecraft texture recoloring tool for Windows. Recolor your textures instantly with no setup required.

---

## Features
- Standalone Windows executable – no Python or extra setup needed  
- Quick recoloring of Minecraft PvP textures  
- Easy-to-use interface  
- Lightweight and fast  

---

## Download

Get the latest version of 5hift for Windows:

[Download 5hift v1](https://github.com/5erb/5hift/releases/download/Release/5hift.exe)
