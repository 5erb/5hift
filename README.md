# 5hift
This is a standalone tool that lets you recolor all PvP-related textures in any Minecraft texture pack. You can change the color of diamond gear, weapons, hearts, fire effects, and the pack icon. The tool shows a live preview of selected items and allows you to pick any custom color. Once done, it outputs a ready-to-use zip of your recolored pack.

It’s easy to use — just upload your texture pack, pick a color, and start. No coding or image editing knowledge required. The tool cleans up temporary files automatically, so you get a clean zip every time.
